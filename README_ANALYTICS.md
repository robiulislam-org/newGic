# 🔒 গুগল-মুক্ত ও সম্পূর্ণ প্রাইভেট অ্যানালিটিক্স ড্যাশবোর্ড সেটআপ গাইড (Supabase)

আপনার অনুরোধ অনুযায়ী, আমরা গুগলকে (Firebase) সম্পূর্ণ বাদ দিয়ে শতভাগ ওপেন-সোর্স ও প্রাইভেট প্ল্যাটফর্ম **Supabase** দিয়ে আপনার অ্যানালিটিক্স সিস্টেম এবং ড্যাশবোর্ড রি-ডিজাইন করেছি। এর ফলে আপনার ওয়েবসাইটের ডেটার সম্পূর্ণ নিয়ন্ত্রণ কেবল আপনার কাছে থাকবে, গুগলের কাছে কোনো তথ্য যাবে না।

এটি সচল করতে মাত্র ৩টি সহজ ধাপ সম্পূর্ণ করুন:

---

## ধাপ ১: Supabase-এ ফ্রি প্রজেক্ট তৈরি করুন
1. ব্রাউজারে [Supabase Website](https://supabase.com/) এ যান এবং আপনার জিমেইল বা গিটহাব অ্যাকাউন্ট দিয়ে লগইন করুন।
2. **"New Project"** বাটনে ক্লিক করুন।
3. আপনার পছন্দমতো কোনো একটি Organization সিলেক্ট করুন (বা নতুন তৈরি করুন)।
4. প্রজেক্টের তথ্যগুলো পূরণ করুন:
   - **Name**: `GIC-Analytics`
   - **Database Password**: আপনার পছন্দের একটি কঠিন পাসওয়ার্ড দিন (এটি মনে রাখুন)।
   - **Region**: `Singapore` (বাংলাদেশের সবচেয়ে কাছের লোকেশন, যা স্পিড বাড়াবে)।
   - **Pricing Plan**: **Free** সিলেক্ট করুন।
5. **Create new project** বাটনে ক্লিক করুন। ডেটাবেস প্রজেক্টটি তৈরি হতে প্রায় ১ মিনিট সময় লাগবে।

---

## ধাপ ২: SQL স্ক্রিপ্ট রান করুন (টেবিল ও সিকিউরিটি সেটআপ)
1. প্রজেক্ট তৈরি শেষ হলে আপনার ড্যাশবোর্ডের বাম পাশের সাইডবার থেকে **"SQL Editor"** আইকনটিতে ক্লিক করুন (আইকনটি দেখতে `>_` এর মতো)।
2. **"+ New query"** বাটনে ক্লিক করে একটি নতুন কুয়েরি ফাইল খুলুন।
3. আপনার প্রজেক্টের মূল ডিরেক্টরিতে থাকা [setup.sql](file:///d:/GIC%20website/setup.sql) ফাইলের সমস্ত কোড কপি করে এখানে পেস্ট করুন।
4. পেস্ট করার পর ডানদিকের নিচে থাকা **"Run"** বাটনে ক্লিক করুন।
5. স্ক্রিনে `Success. No rows returned` দেখালে বুঝবেন আপনার ডেটাবেস টেবিল ও ড্যাশবোর্ডের নিরাপত্তা কুয়েরি সফলভাবে সেটআপ হয়ে গেছে!

*(নোট: আপনার ড্যাশবোর্ড পাসওয়ার্ড পরিবর্তন করতে চাইলে setup.sql ফাইলের ৪১ নম্বর লাইনে `expected_password text := 'gicadmin786';` পরিবর্তন করে আপনার পাসওয়ার্ড বসিয়ে নিতে পারেন)*

---

## ধাপ ৩: এপিআই কি (API Credentials) কপি ও পেস্ট করুন
1. প্রজেক্ট সেটিংস এ যাওয়ার জন্য বাম পাশের সাইডবারের নিচে থাকা গিয়ার আইকনে (⚙️ Project Settings) ক্লিক করুন।
2. সাব-মেনু থেকে **API** অপশনে যান।
3. এখানে আপনি দুটি জিনিস পাবেন:
   - **Project URL**: এটি দেখতে `https://xxxxxx.supabase.co` এর মতো হবে।
   - **Project API Keys (anon public)**: এটি দেখতে অনেকগুলো অক্ষরের একটি লম্বা কি (Key)।
4. আপনার কম্পিউটার থেকে প্রজেক্টের [gic-config.js](file:///d:/GIC%20website/gic-config.js) ফাইলটি ওপেন করুন এবং এগুলো বসিয়ে দিন। উদাহরণস্বরূপ:

```javascript
export const supabaseUrl = "https://xxxxxx.supabase.co";
export const supabaseAnonKey = "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJpc3MiOiJzdXBhYmFzZSIsInJlZiI6ImJyc...";

// ড্যাশবোর্ডে ঢোকার পাসওয়ার্ড (যা setup.sql এ বসিয়েছেন, ডিফল্ট: gicadmin786)
export const adminPassword = "gicadmin786";
```

৫. ফাইলটি সেভ করে দিন।
